# Imágenes a pdf

> Utilidad para componer múltiples imágenes en un documento pdf.

## Version
2023-01-12 11:04:56

## Modo de uso
1. Configurar rutas y seteos dentro de `cfg.yml`.
2. Ejecutar `imagenes_a_pdf.py`.

## Requerimientos
+ imagemagick
+ python3
  + pypdf2
  + wand
  + pillow
